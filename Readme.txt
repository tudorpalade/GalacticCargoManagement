Am folosit o librarie JSON,downloadata de pe pagina: http://www.java2s.com/Code/Jar/j/Downloadjsonsimple111jar.htm
Pt a o downloada ,click pe Download json-simple-1.1.jar de pe pagina de mai sus.

Pt a include libraria:
In IDEE:
 	1. Click stanga pe proiect , select Properties
	2. Deschidem Java Build Path 
	3. Selectam Libraries 
	4. Selectam Add External JARs
	5. Localizam fisierul JAR( downloadat mai sus ).
	6. Click OK .
Acum putem folosi fisierul JAR .

-----------------------------------------------------

Pt a rula executabilul JAR : GalacticCargoManagement.jar :

	Deschidem command promt
	Narvigam in fisierul in care se afla jar-ul.
	Scriem java -jar GalacticCargoManagement.jar si apasam OK.


------------------------------------------------------

Fisierele .json, trebuie copiate in interiorul folderului proiectului.
